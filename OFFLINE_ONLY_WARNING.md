# Offline-Only Warning

BakkesSwap is a local-only and offline-only visual swap tool.

- do not use it online
- do not use it to modify server inventory
- do not use it to unlock items
- do not use it as an anti-cheat bypass tool
- do not use it with runtime hooks or overlays in this beta

This beta package must be tested only against fake or copied local data.

During beta validation:

- do not point it at a real Rocket League install
- do not include real `.upk` files in tests
- do not include inventory dumps or personal workspace state in redistributed folders

Restore your original local files before any normal or online play.
